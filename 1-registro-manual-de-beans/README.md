##Proyecto 01 — Registro manual de beans
Anotaciones: @Configuration, @Bean
Consigna:
Sin usar @Component ni @Service,
crea una clase @Configuration con métodos @Bean. 
Registrar una clase Calculadora y una clase ProcesadorDePagos. 
Inyectar la Calculadora dentro del ProcesadorDePagos pasándola 
como parámetro del método @Bean.

Imprimir desde el constructor de cada clase para ver el orden de creación.
