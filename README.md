Micro Proyectos Spring Boot

**mini proyectos** para practicar  conceptos clave de **Spring Boot**, **JPA**, **HTMX**
---

## **Objetivos**
- Entender el **ecosistema Spring Boot** (inyección de dependencias, configuración, etc.).
- Practicar **JPA/Hibernate** (entidades, repositorios, relaciones).
- Explorar **anotaciones**
- Experimentar con **HTMX**
- Aprender **estructuras de proyectos Mavenn
