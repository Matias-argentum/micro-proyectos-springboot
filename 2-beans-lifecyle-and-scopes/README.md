##Proyecto 02 — ciclos de vida y scopes
Anotaciones: @Scope, @PostConstruct, @PreDestroy
Consigna:
Creá un bean con @PostConstruct y @PreDestroy para observar cuándo se inicializa y destruye. 
Luego cambiá entre @Scope("singleton") y 
@Scope("prototype") y pedí el bean varias veces 
desde el contexto. 
Registrá en consola cuántas instancias se crean.
